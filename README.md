🎬 Netflix Data Analysis Dashboard (Excel)

This project is an Excel-based dashboard built using a dataset of movies and TV shows. It focuses on exploring trends such as content distribution, ratings, release years, and country-wise production.

📊 Dataset Overview

The dataset contains information about Netflix titles with the following columns:
show_id – Unique ID for each title
type – Movie or TV Show
title – Name of the content
director – Director name
cast – Actors involved
country – Country of origin
date_added – Date added to Netflix
release_year – Year of release
rating – Content rating (PG, TV-MA, etc.)
duration – Duration or number of seasons
listed_in – Genre/category
description – Short summary

📈 Dashboard Features

The Excel dashboard includes multiple interactive visuals:

Content Type Distribution
Comparison between Movies and TV Shows
Release Year Trend
Shows how content production has increased over time
Ratings Analysis
Distribution of titles across ratings like TV-MA, PG-13, etc.
Top Directors
Directors with the highest number of titles
Country-wise Content
Contribution of different countries
Combined Charts
Comparison of Movies vs TV Shows across ratings and years

🛠 Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers for filtering

📌 Key Insights
Movies dominate the dataset compared to TV Shows
A sharp increase in content is visible after 2015
TV-MA is the most common rating
United States contributes the highest number of titles
Some directors appear multiple times, showing frequent collaborations

🚀 How to Use

Download the Excel file
Open in Microsoft Excel
Use slicers (filters) to explore different views
Interact with charts to analyze trends
